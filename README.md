# IM

##### 简介

这是一个IM即时通信的服务端，相应的API接口可以查看API文档，目前支持的功能有登录、注册、添加好友、创建群聊、添加群聊、私聊、群聊、支持文字和图片的发送。朋友圈的发布、点赞和评论、以及个人信息修改等。

##### 服务端部署方法

需要安装mongodb和go语言环境，执行go run main.go运行代码即可开启服务端。
